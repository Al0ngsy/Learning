# Practical Agile and Scrum Exercises

Use these exercises to turn theory into team habits.

## Exercise 1: User Story Writing

Goal: write clear user stories with acceptance criteria.

Template:
- As a <user>, I want <capability>, so that <benefit>.
- Acceptance Criteria:
  - Given <context>, when <action>, then <expected outcome>.

Task:
- Write 5 user stories for a simple app (e.g., task tracker).
- Add at least 2 acceptance criteria per story.

## Exercise 2: Backlog Prioritization

Goal: practice value-based prioritization.

Task:
- Create a backlog of 12 items.
- Rank by business value, risk reduction, and effort.
- Mark top 5 as candidates for the next sprint.

## Exercise 3: Sprint Planning Simulation

Goal: define a realistic sprint goal and commitment.

Task:
- Assume a 2-week sprint and fixed team capacity.
- Select stories from backlog and break into tasks.
- Set a sprint goal and identify dependencies.

## Exercise 4: Daily Scrum Practice

Goal: improve transparency and coordination.

For 5 consecutive days, answer:
- What did we complete since last sync?
- What will we complete before next sync?
- What blockers need help?

## Exercise 5: Sprint Retrospective

Goal: drive continuous improvement.

Use Start/Stop/Continue:
- Start: one new practice to try next sprint.
- Stop: one inefficient behavior to remove.
- Continue: one successful habit to preserve.

Create 3 concrete actions with owners and due dates.
